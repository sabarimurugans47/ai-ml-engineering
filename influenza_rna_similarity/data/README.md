# 🧬 Influenza RNA Dataset

This folder contains **Influenza A virus RNA sequence segments** used for similarity analysis in this project.

Influenza A viruses have **8 segmented RNA genomes**, each responsible for different viral functions.

---

# 📂 Dataset Files

| File | Segment Name | Description |
|-----|-------------|-------------|
| HA.fa | Hemagglutinin | Surface protein responsible for viral entry into host cells |
| NA.fa | Neuraminidase | Surface protein involved in virus release from infected cells |
| NP.fa | Nucleoprotein | Encapsulates viral RNA genome |
| NS.fa | Non-structural protein | Regulates host immune response |
| M.fa | Matrix protein | Maintains viral structure |
| PA.fa | Polymerase acidic protein | Part of viral RNA polymerase complex |
| PB1.fa | Polymerase basic protein 1 | RNA replication enzyme |
| PB2.fa | Polymerase basic protein 2 | Helps initiate RNA transcription |

---

# 🧪 Data Format

All files are stored in **FASTA format (.fa)**.


---

# 🔬 Usage in This Project

These RNA sequences are used to:

1. Extract sequence data
2. Apply **n-gram tokenization**
3. Convert sequences into numerical vectors
4. Calculate **cosine similarity**
5. Compare genetic similarity between viral sequences

RNA sequence data obtained from public bioinformatics databases such as:
