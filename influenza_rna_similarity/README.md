# 🧬 Influenza RNA Sequence Similarity Model

This project analyzes **similarity between Influenza RNA sequences** using Natural Language Processing techniques.  
The goal is to measure how similar or different RNA sequences are by applying **n-gram tokenization and cosine similarity**.

The model helps demonstrate how machine learning techniques can be applied to **bioinformatics sequence analysis**.

---

# 📌 Project Overview

Influenza viruses contain segmented RNA genomes.  
Understanding sequence similarity can help analyze genetic relationships between different viral strains.

This project:

• Processes RNA sequence data  
• Converts sequences into numerical features using **n-grams**  
• Computes similarity using **cosine similarity**  
• Compares Influenza RNA sequences with other viral sequences

# ⚙️ Methods Used

This project applies several techniques from **machine learning and NLP**:

- N-gram tokenization for sequence representation
- Cosine similarity for similarity measurement
- Vectorization of biological sequences
- Basic preprocessing for RNA data
